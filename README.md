🔢 Konversi Angka ke Bilangan Romawi dengan Excel
Deskripsi
Proyek ini berisi pengolahan data menggunakan Microsoft Excel untuk mengonversi angka ke bilangan Romawi dan sebaliknya. Data yang digunakan mencakup angka seperti 1, 4, 18, 124, hingga 2030.

Tools & Pendekatan
- Microsoft Excel
- Logika konversi angka ke Romawi
- Pengolahan data berbasis rumus sederhana

Hasil
- Angka berhasil dikonversi ke bentuk Romawi
- Data dapat ditampilkan dalam dua format (angka & Romawi)
- Memahami pola bilangan Romawi melalui Excel

Tujuan Proyek
- Melatih logika konversi angka
- Memahami sistem bilangan Romawi
- Mengasah kemampuan pengolahan data di Excel

Insight
Konversi angka ke Romawi membantu memahami pola matematika klasik dan bagaimana logika sederhana dapat diterapkan dalam pengolahan data menggunakan Excel.

Penutup
Proyek ini merupakan latihan dasar dalam pengolahan data berbasis logika matematika menggunakan Excel.
